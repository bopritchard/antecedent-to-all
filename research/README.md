# Research Notes

This folder contains research notes, findings, and analysis organized by topic or article.

## Organization

- Create files by topic: `topic-name-research.md`
- Or by article: `article-title-research.md`
- Include source citations
- Note key findings and how they support arguments

## Format Example

```markdown
# Research: [Topic/Article Title]

## Key Findings
- Finding 1 with source
- Finding 2 with source

## Sources
- [Source 1](URL)
- [Source 2](URL)

## Notes
[Additional context, questions, or analysis]
```
