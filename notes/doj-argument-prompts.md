## Argument-Building Tips

- Start with constitutional text (Article II)
- Use originalist methodology
- Address counterarguments directly
- Use historical examples
- Connect to core themes (unitary executive, democratic accountability)
- Build logically from premises to conclusion

## Connection to Core Themes

### Original Authority
- Arguments should start with Constitution's text
- Use original intent and understanding
- Distinguish original authority from modern construction

### Unitary Executive
- Arguments should support unitary executive principle
- Show how independence conflicts with unitary executive
- Demonstrate constitutional requirement for executive control

### Democratic Accountability
- Arguments should emphasize popular sovereignty
- Show how independence undermines electoral accountability
- Connect accountability to democratic legitimacy

### Constitutional Drift
- Arguments should identify how independence represents drift
- Show how practice has diverged from original understanding
- Critique modern construction that lacks constitutional basis
