# Quotes Collection

This folder contains quotes organized by topic or article for integration into articles.

## Organization

- Create files by topic: `topic-name-quotes.md`
- Or by article: `article-title-quotes.md`
- Include full attribution: author, source, date
- Note context for each quote

## Format Example

```markdown
# Quotes: [Topic/Article Title]

## Quote 1
> "The quote text here."
> 
> — Author Name, *Source Title* (Date)

Context: [How this quote supports the argument]

## Quote 2
> "Another quote."
> 
> — Author Name, *Source Title* (Date)

Context: [How this quote supports the argument]
```
